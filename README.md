# i2cscan
Simple tool to scan for i2c devices on the bus. Tested on the [Redpitaya]  plattform.

### Installation

```sh
$ git clone https://github.com/noah95/i2cscan
$ cd i2cscan
$ make
```

### Run
Copy the binary i2cscan to your Redpitaya and run
```sh
$ chmod +x i2cscan
$ ./i2cscan
```
